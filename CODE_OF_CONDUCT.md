# Code of Conduct

BeamKit follows the Contributor Covenant Code of Conduct.

## Our Standard

Contributors are expected to communicate respectfully, focus on the work, and make space for clinical, technical, research, and open-source perspectives.

Unacceptable behavior includes harassment, discriminatory language, threats, doxxing, and publishing private or sensitive information such as patient data or credentials.

## Enforcement

Maintainers may remove comments, reject contributions, or block contributors who violate this code of conduct.

When the project is hosted publicly, reports should be sent through the repository maintainer contact or GitHub moderation tools.
