# Sample Rule Packs

These rule packs are PHI-free examples for BeamKit CLI demos, CI server smoke tests, and rule-pack authoring workflows.

| Pack | Manifest | Notes |
| --- | --- | --- |
| Head and Neck | [head-neck-v1/beamkit-rule-pack.json](head-neck-v1/beamkit-rule-pack.json) | Complete synthetic sample with approval metadata and head-and-neck regression cases. |
| Lung SBRT | [lung-sbrt-v1/beamkit-rule-pack.json](lung-sbrt-v1/beamkit-rule-pack.json) | Draft starter pack generated by `rule-pack new`. |
| Prostate | [prostate-v1/beamkit-rule-pack.json](prostate-v1/beamkit-rule-pack.json) | Draft starter pack generated by `rule-pack new`. |
| Brain SRS | [brain-srs-v1/beamkit-rule-pack.json](brain-srs-v1/beamkit-rule-pack.json) | Draft starter pack generated by `rule-pack new`. |
| Breast | [breast-v1/beamkit-rule-pack.json](breast-v1/beamkit-rule-pack.json) | Draft starter pack with future regression-case placeholders. |
| Palliative | [palliative-v1/beamkit-rule-pack.json](palliative-v1/beamkit-rule-pack.json) | Draft starter pack with future regression-case placeholders. |

Run authoring checks:

```bash
dotnet run --project src/BeamKit.Cli -- rule-pack doctor \
  --rule-pack samples/rule-packs/head-neck-v1/beamkit-rule-pack.json
```

Generate a new starter:

```bash
dotnet run --project src/BeamKit.Cli -- rule-pack new \
  --disease-site lung-sbrt \
  --institution "Example Clinic" \
  --owner Physics \
  --output artifacts/rule-packs/lung-sbrt-v1
```

See [docs/rule-pack-authoring.md](../../docs/rule-pack-authoring.md).
